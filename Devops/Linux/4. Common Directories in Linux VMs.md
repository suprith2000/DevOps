| Directory        | Description                                                                  |
| ---------------- | ---------------------------------------------------------------------------- |
| `/`              | Root directory. The top-level directory in Linux. All paths start from here. |
| `/bin`           | Essential user binaries (commands like `ls`, `cp`, `mv`, `bash`).            |
| `/boot`          | Files needed for booting (e.g., Linux kernel, GRUB config).                  |
| `/dev`           | Device files (e.g., `/dev/sda`, `/dev/null`, etc.).                          |
| `/etc`           | Configuration files (e.g., `/etc/passwd`, `/etc/hosts`).                     |
| `/home`          | Home directories for users (e.g., `/home/suprith`).                          |
| `/lib`, `/lib64` | Essential shared libraries used by binaries in `/bin` and `/sbin`.           |
| `/media`         | Temporary mount points for removable media (USBs, CDs).                      |
| `/mnt`           | Used to mount temporary filesystems.                                         |
| `/opt`           | Optional software/packages (e.g., third-party apps).                         |
| `/proc`          | Virtual filesystem exposing kernel and process info. (e.g., `/proc/cpuinfo`) |
| `/root`          | Home directory of the `root` user.                                           |
| `/run`           | Runtime variable data (like PIDs, sockets, etc.).                            |
| `/sbin`          | System binaries (e.g., `fsck`, `reboot`) â€” essential for booting/repairing.  |
| `/srv`           | Data for services like HTTP, FTP. Rarely used directly.                      |
| `/sys`           | Exposes info about devices, drivers, kernel (sysfs).                         |
| `/tmp`           | Temporary files. Cleared on reboot.                                          |
| `/usr`           | Secondary hierarchy for user programs, includes `/usr/bin`, `/usr/lib`, etc. |
| `/var`           | Variable data like logs, mail, spool files.                                  |
