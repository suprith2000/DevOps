### üïπÔ∏è Basic Vim Modes

Vim has **modes**, unlike regular editors:

|Mode|Description|
|---|---|
|`Normal`|Default mode (used for navigation & commands)|
|`Insert`|You type text in this mode (`i` to enter)|
|`Visual`|Select text (`v` to enter)|
|`Command`|Used to run commands like `:w` (write/save), `:q` (quit)|

---

### üìù Common Vim Commands

| Task                | Command                          |
| ------------------- | -------------------------------- |
| Enter Insert Mode   | `i`                              |
| Save file           | `:w`                             |
| Quit Vim            | `:q`                             |
| Save and quit       | `:wq` or `ZZ`                    |
| Quit without saving | `:q!`                            |
| Move cursor         | Arrow keys or `h`, `j`, `k`, `l` |
| Delete line         | `dd`                             |
| Copy line           | `yy`                             |
| Paste               | `p`                              |
| Undo                | `u`                              |
| Redo                | `Ctrl + r`                       |

---

### üí° Tips

- Vim has a steep learning curve but is very efficient once mastered.
- You can exit if stuck by pressing `Esc` then typing `:q!` + `Enter`.
