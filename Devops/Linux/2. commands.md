mv (Move)
You can rename files like this:
$ mv oldfile newfile
move a file to a different directory:
$ mv file2 /home/pete/Documents
move more than one file:
$ mv file_1 file_2 /somedirectory
rename directories as well:
$ mv directory1 directory2
use the -i flag to prompt you before overwriting anything.
$mv -i directory1 directory2
mv a file to overwrite the previous one. You can also make a backup of that file and it will just rename the old version with a ~
$ mv -b directory1 directory2

rm (Remove)
to delete files and directories.
$ rm file1
absolutely remove a bunch of files.
-f or force option tells rm to remove all files whether they are write protected or not, without prompting the user
$ rm -f file1
-i flag like many of the other commands, will give you a prompt on whether you want to actually remove the files or directories.
$ rm -r directory
You can remove a directory with the rmdir command.
$ rmdir directory
